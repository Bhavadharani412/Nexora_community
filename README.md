## ✨ Overview

Nexora is a futuristic community platform designed with a premium visual experience and smooth interactions.

The project focuses on:
- modern UI/UX
- cinematic layouts
- smooth animations
- responsive design
- scalable architecture
- community-driven ecosystem

---

# 🚀 Features

- ⚡ Next.js 14 App Router
- 🎨 Tailwind CSS
- ✨ Framer Motion animations
- 🌌 Cinematic futuristic UI
- 📱 Fully responsive
- 🧩 Modular component architecture
- 🧠 Smooth navigation tracking
- 🎯 Contributor & partner showcases
- 🔗 Social integrations
- 🎭 Glassmorphism effects
- 🖱️ Custom cursor support

# ⚙️ Installation

## 1. Clone the repository

```bash
git clone https://github.com/P-r-o-f-e-s-s-o-r/Nexora_community.git
```

---

## 2. Navigate to project

```bash
cd Nexora_community
```

---

## 3. Install dependencies

```bash
npm install
```

---

## 4. Start development server

```bash
npm run dev
```
